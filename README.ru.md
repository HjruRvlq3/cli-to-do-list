# CLI To-Do List

Консольное приложение на Go для управления списком задач.
Позволяет добавлять, просматривать и удалять задачи из командной строки.

---

## Функционал

- Добавление новой задачи
- Просмотр всех задач
- Удаление задачи по номеру
- Работает в цикле до команды
