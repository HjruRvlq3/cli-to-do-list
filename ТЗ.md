## Функциональные требования

### 1. Работа с пунктами (Task Management)

- Создание новой задачи
- Отметка выполнения задачи
- Отмена отметки выполнения задачи
- Редактирование текста задачи
- Удаление задачи
- Удаление всех выполненных задач
- Сохранение задач между запусками

### 2. Выводы (Output)

- Вывод всех задач
- Вывод невыполненных задач
- Вывод выполненных задач
- Поддержка команд:
  - `help` / `h` — вывод справки по командам
  - `exit` — завершение работы программы

### 3. Ввод (Input)

- Обработка команд пользователя
- Валидация номеров задач и введённых команд
- Ввод текста задачи
- Подтверждение действий при удалении:
  - одной невыполненной задачи
  - всех выполненных задач

## Взаимодействие пакетов

1. **main.go** запускает цикл программы и обрабатывает глобальные команды (`exit`, `help`)  
2. **input.go** получает и валидирует команды пользователя  
3. **task.go** выполняет действия с задачами (создание, редактирование, отметка, удаление)  
4. **output.go** вывод в консоль  
5. **taskBase.go** сохраняет и загружает задачи, обеспечивая их сохранность между запусками

## Требования к пользовательскому опыту

- Простое добавление, редактирование и удаление задач
- Понятный вывод статуса задач `[✓]` для выполненных, `[ ]` для невыполненных
- Подтверждение при удалении всех выполненных задач или одной не выполненной задачи
